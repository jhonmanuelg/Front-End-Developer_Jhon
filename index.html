Projetos

Netflix Landing Html Css.

O projeto Netflix Landing é uma página web que busca replicar o layout e estilo visual da página inicial da plataforma de streaming Netflix. Esse projeto utiliza HTML e CSS para criar uma interface atrativa e responsiva, semelhante à página real da Netflix.

Operações CRUD do JS.

O projeto Operações CRUD (Create, Read, Update, Delete) com HTML, CSS e JavaScript é uma ótima oportunidade para praticar habilidades de desenvolvimento web. Ele envolve a manipulação de dados, interação do usuário e design de interface. Além disso, é uma base fundamental para a construção de aplicativos mais complexos que envolvem o gerenciamento de informações.

Landing Page Bootstrap 5.

O projeto Landing Page Bootstrap 5 é um projeto de desenvolvimento web que envolve a criação de uma página inicial atraente e responsiva utilizando o framework Bootstrap 5. O Bootstrap 5 é uma biblioteca de código aberto amplamente utilizada para o desenvolvimento de interfaces de usuário responsivas e modernas.

Cadastro IMC Html Css Js.

O projeto "Cadastro do usuário IMC" é uma aplicação web que permite aos usuários calcular seu Índice de Massa Corporal (IMC) e armazenar as informações em um cadastro pessoal. O IMC é uma medida que avalia a relação entre o peso e a altura de uma pessoa, fornecendo uma indicação do seu estado de saúde em termos de peso corporal.

Projeto no React.

O projeto no React de análise de desempenho das vendas é uma aplicação web desenvolvida para ajudar empresas a monitorar e analisar o desempenho das suas vendas. O objetivo é fornecer insights valiosos sobre as vendas realizadas, permitindo que a empresa tome decisões informadas para melhorar seus resultados.
